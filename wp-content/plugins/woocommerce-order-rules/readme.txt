== Description ==

WooCommerce Order Rules is an extension which allows you to automatically filter out the orders based on a filter which you define by colouring the row of the order to a HEX colour you assign.

== Installation ==

1. Upload the entire 'woocommerce-order-rules' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
